# FitConnect - Project Instructions

## Overview
Platform connecting Personal Trainers (Professors) and Students. Built with Next.js 14, Prisma, PostgreSQL.

## Auth
- Professor: `john@doe.com` / `johndoe123`
- Student: `aluno@teste.com` / `aluno123`

## Visual Style
- Primary: Indigo (#4f46e5), Accent colors
- Font: Display font for headings, clean sans-serif
- Cards with shadow-md, rounded-xl
- Framer Motion animations on landing page

## Plan Structure
- **free**: 2 students, basic features
- **pro10**: 10 students, R$39.90/month, R$29.90/annual
- **pro50**: 50 students, R$59.90/month, R$49.90/annual (most popular)
- **pro100**: 100 students, R$89.90/month, R$79.90/annual
- Centralized in `lib/plans.ts`

## Key Architecture
- `lib/plans.ts` - Plan definitions and limits
- `lib/stripe.ts` - Stripe client (lazy init via Proxy)
- `lib/notifications.ts` - Email notification helper
- `lib/aws-config.ts` + `lib/s3.ts` - S3 file upload
- Stripe integration: checkout, webhook, billing portal
- Email notifications: student invite, workout assigned, payment confirmation

## Database
- Professor has: stripeCustomerId, stripeSubscriptionId, stripeBillingCycle, plan, maxStudents
- Exercises have: setsConfig (Json), warmupConfig (Json), media fields
- ExerciseLog has: setsLog (Json), warmupLog (Json)

## Important Patterns
- Calendar uses local dates (not UTC) to avoid timezone issues (Brazil UTC-3)
- Exercise config backward-compatible: parseExercise handles both flat + JSON formats
- Nav items defined locally in each component (professor & aluno)
- Professor nav includes: Dashboard, Alunos, Treinos, Meu Plano
- Aluno nav includes: Dashboard, Meus Treinos, Calendário, Histórico
