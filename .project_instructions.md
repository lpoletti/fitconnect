# FitConnect - Project Instructions

## Overview
FitConnect é uma plataforma MVP que conecta Personal Trainers (Professores) e Alunos para gestão de treinos. Modelo freemium com plano grátis para até 2 alunos ativos.

## Visual Style
- **Primary color**: Emerald green (HSL 160 84% 39%)
- **Accent color**: Orange (HSL 25 95% 53%)
- **Fonts**: DM Sans (body), Plus Jakarta Sans (display headings), JetBrains Mono (monospace)
- **Theme**: Light mode default, dark mode supported
- **Design language**: Clean, modern, card-based with rounded corners and subtle shadows

## Technical Architecture
- **Auth**: NextAuth.js v4 with Credentials provider, JWT strategy
- **DB**: PostgreSQL via Prisma ORM
- **User types**: 'professor' | 'aluno' stored in User.userType
- **Session**: extends with userId, userType, professorId, studentId

## Page Structure
### Public Pages
- `/` - Landing page with hero, benefits, how it works, CTA
- `/login` - Unified login
- `/signup` - Signup with professor/aluno toggle

### Professor (protected, middleware-guarded)
- `/professor/dashboard` - Stats, student list
- `/professor/alunos` - Manage students, add by email, accept/reject/deactivate
- `/professor/treinos` - Workout template CRUD
- `/professor/treinos/novo` - Create workout
- `/professor/treinos/[id]` - View workout details
- `/professor/treinos/[id]/editar` - Edit workout
- `/professor/alunos/[id]/atribuir-treino` - Assign workout to student

### Aluno (protected, middleware-guarded)
- `/aluno/dashboard` - Active workouts, professor, history
- `/aluno/treinos` - List active workouts
- `/aluno/treinos/[id]` - Execute workout (mark exercises, log weights)
- `/aluno/historico` - Workout history with filters

## Business Rules
- Free plan: max 2 active students per professor
- Validated on backend (POST /api/professor/students, PATCH status to active)
- Workout templates are unlimited
- Student-professor auto-link when student signs up with invited email

## Database
10 models: User, Professor, Student, StudentProfessorLink, WorkoutTemplate, WorkoutTemplateExercise, AssignedWorkout, AssignedWorkoutExercise, WorkoutLog, ExerciseLog
